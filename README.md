# 🏋️ Fitness Tracker

A simple web app to log workouts and track progress. Built with plain HTML, CSS,
and JavaScript — no installation or build tools required.

## Features

- Log a workout (exercise, duration, date, optional notes)
- See total workouts and weekly totals at a glance
- Delete entries
- Your data is saved automatically in your browser (localStorage)

## How to run it

Just open `index.html` in your web browser — double-click the file, or drag it
into a browser window. That's it.

## Project structure

| File         | What it does                                  |
| ------------ | --------------------------------------------- |
| `index.html` | The page structure (headings, form, list)     |
| `style.css`  | How the page looks (colors, layout, spacing)  |
| `app.js`     | The behavior (saving, loading, updating)      |
